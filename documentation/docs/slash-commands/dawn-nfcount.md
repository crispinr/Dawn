# dawn-nfcount

Gets **data** on the **total supply** of NFTs in a **collection**.
## Syntax

`/dawn-nfcount <collection_address>`

## Parameters

| Name               | Explanation                                                            | Optional
| ------------------ | ---------------------------------------------------------------------- | ----------
| collection_address | String of collection address to filter by	                            | false
