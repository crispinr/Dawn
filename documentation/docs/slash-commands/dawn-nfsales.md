# dawn-nfsales

Gets **sales data** for any **NFT** across **multiple marketplaces**.

## Syntax

`/dawn-nfsales <collection_address>`

## Parameters

| Name               | Explanation                               | Optional     |
| ------------------ | ----------------------------------------- | ------------ |
| collection_address | String of collection address to filter by | false        |
