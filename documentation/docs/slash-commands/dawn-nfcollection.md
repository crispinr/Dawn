# dawn-nfcollection

Gets **data** for the specified **NFT collection**.

## Syntax

`/dawn-nfcollection <collection_address>`

## Parameters

| Name               | Explanation                                                            | Optional
| ------------------ | ---------------------------------------------------------------------- | ----------
| collection_address | String of collection address to filter by	                            | false
