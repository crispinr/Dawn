# dawn-nfcollection

Gets **data** for the specified NFT **collection**.

## Syntax

`/dawn-nfcollection <collection_address>`

## Parameters

| Name               | Explanation                                                            |
| ------------------ | ---------------------------------------------------------------------- |
| collection_address | Enter the collection address to data of all the NFTs in the collection |
