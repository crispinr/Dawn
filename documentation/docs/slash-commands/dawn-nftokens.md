# dawn-nftokens

Gets **data** on a group of tokens based on **collection address**.

## Syntax

`/dawn-nftokens <collection_address>`

## Parameters

| Name               | Explanation                               | Optional     |
| ------------------ | ----------------------------------------- | ------------ |
| collection_address | String of collection address to filter by | false        |
